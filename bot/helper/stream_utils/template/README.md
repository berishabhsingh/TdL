# Stream Templates

This directory contains HTML templates used for the web interface when streaming media directly from the bot.

## Purpose

When a user opens a streaming link generated by the bot in their web browser, the server defined in `stream_utils/web_services.py` responds by rendering one of these HTML templates. These templates provide a web-based media player (often HTML5 `<video>` or `<audio>` tags) so users can watch or listen to the file without needing a dedicated external media player like VLC.

## Modification

If you wish to change the look and feel of the web player (e.g., adding custom CSS, a logo, or using a different open-source video player library like Video.js), you can modify the template files located in this directory. The Python backend will inject the actual streaming URL into these templates before serving them.
