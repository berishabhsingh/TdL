# qBittorrent

This directory contains configuration files and setup details for integrating qBittorrent with the LeechxTool bot.

## Purpose

The bot utilizes qBittorrent to handle torrent downloads. It communicates with the local qBittorrent client, adding torrents, managing priorities, and monitoring download progress.

## Structure

The `qBittorrent/config/` directory includes essential configuration files such as `qBittorrent.conf`. These settings determine how qBittorrent operates, including network settings, connection limits, download behavior, and web UI integration for external interactions.
